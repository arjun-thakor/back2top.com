Privacy Policy

Last updated: January 14, 2026

Overview

Jump to Top Button ("the extension") is committed to protecting your privacy. This extension does not collect, store, or transmit any personal data.

Data Collection

The extension does NOT collect any of the following:

  • Personal information
  • Browsing history
  • Cookies or tracking data
  • Analytics or usage statistics
  • Any other user data

Data Storage

The extension stores user preferences locally using Chrome's sync storage API. This includes:

  • Button position (bottom-right, bottom-left, etc.)
  • Button size
  • Scroll threshold
  • Theme preference (dark/light)
  • Reduced motion setting

These settings are stored locally in your browser and synced across your devices using Chrome's built-in sync functionality. This data never leaves your browser or Chrome account.

Permissions

The extension requests the following permissions:

  • storage: To save your preferences locally
  • activeTab: To respond to keyboard shortcuts on the active tab
  • host_permissions (<all_urls>): To inject the floating button on all websites you visit

These permissions are used solely for the extension's functionality and not for data collection.

Third-Party Services

The extension does not use any third-party services, analytics, or tracking tools.

Remote Code

The extension does not load or execute any remote code. All JavaScript is bundled with the extension package.

Changes to This Policy

Any changes to this privacy policy will be updated on this page and reflected in the "Last updated" date above.

Contact

If you have any questions about this privacy policy, please contact us through the GitHub repository.
